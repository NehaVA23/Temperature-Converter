# Temperature-Converter

 🌡️ Temperature Converter Web App

A small Flask-based web application that allows users to convert temperatures between Celsius and Fahrenheit. Created as part of an internship task focused on conditional logic and user interaction through a web interface.

 💡 Features

- Convert Celsius to Fahrenheit and vice versa
- User inputs temperature and selects conversion type
- Instant result displayed on the page
- Simple and responsive UI using HTML & CSS
 📁 Project Structure

TemperatureConverter/

├── templates/
│   └── index.html
├── static/
│   └── style.css

 How to Run

1. Install Flask (if needed):
``
pip install flask

2. Start the app:

python app.py

3. Open in browser:


[http://localhost:5000](http://localhost:5000)
